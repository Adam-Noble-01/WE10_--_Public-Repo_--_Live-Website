#### Global Favicon Files Library Index

- Note Prefer SVG format for favicons as it will not pixelate on high res screens.
- If SVG is not supported, use PNG format.
- If PNG is not supported, use ICO format.

```favicon_library_index
/assets/AD05_-_LIBR_-_Common_-_Icons-and-favicons
    AD05_01_-_NA_Favicon_-_SVG-h50mm.svg       <-- Use this as the primary default core favicon as it will not pixelate 
    AD05_02_-_NA_Favicon_-_PNG-h2000px.png     <-- This is used for High resolution square aspect ration profile images.
    AD05_03_-_NA_Favicon_-_PNG-h1000px.png     <-- This is used for Medium resolution square aspect ration profile images.
    AD05_04_-_NA_Favicon_-_PNG-h512px.png      <-- This is used for browsers that do not support modern SVG format, use as a fallback.
    AD05_05_-_NA_Favicon_-_PNG-h192px.png      <-- This is a fallback icon uf the 512px version does not load.
    AD05_06_-_NA_Favicon_-_PNG-h32px.png       <-- This is the absolute fallback icon uf the SVG, 512px, 192px versions fail to load.
    AD05_07_-_NA_Favicon_-_ICO-h32px.ico       <-- This is the absolute fallback icon for browsers that do not support the other formats.
```
