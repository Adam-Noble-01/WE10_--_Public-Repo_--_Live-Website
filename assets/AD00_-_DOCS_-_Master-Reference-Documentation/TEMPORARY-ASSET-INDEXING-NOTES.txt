# FAVICON SETUP FOR BOOKMARK ICONS
-----------------------------------------------------------------------------------------------------------------------

## What Is a Favicon?
A *favicon* is a small icon (typically 16×16 or 32×32) shown:
- In browser tabs
- On bookmarks/favourites bars
- When creating website shortcuts

The square icons you're asking about are likely generated from properly implemented **favicon metadata** on your site.

---

## Recommended File Format

### Use `.ico` (Best Compatibility)  
Browsers still prefer `.ico` files for classic favicons, particularly for legacy support.

### PNG is Also Fine (Modern Browsers)  
You can use PNG too, especially for higher resolution versions (e.g. 192×192 or 512×512) – used by mobile browsers or pinned apps.

**Recommendation:**  
Use both `.ico` (for classic favicon) and `.png` (for high-res/modern) and reference them correctly.

## HTML Code To Include in `<head>`

Insert this in your HTML's `<head>` tag:

```html
<!-- Classic Favicon -->
<link rel="icon" type="image/x-icon" href="/assets/icons/favicon.ico">

<!-- Modern Browsers -->
<link rel="icon" type="image/png" sizes="32x32" href="/assets/icons/favicon-32.png">
<link rel="icon" type="image/png" sizes="192x192" href="/assets/icons/favicon-192.png">
```

You can also add this if you want to cover Apple devices or Android homescreens:

```html
<!-- Apple Touch Icon -->
<link rel="apple-touch-icon" sizes="180x180" href="/assets/icons/apple-touch-icon.png">

<!-- Android Manifest (Optional) -->
<link rel="manifest" href="/manifest.json">
```

---

## Creating a `.ico` File

Use any of these free tools:
- [https://realfavicongenerator.net/](https://realfavicongenerator.net/)
- [https://favicon.io/](https://favicon.io/)

They allow uploading a PNG or SVG, and will generate `.ico`, `.png`, and HTML tags for you.

## Why Your Bookmarks Show Square Icons

In your screenshot, “Portal Links” and “Coding Scripts” have favicons with a **Material Design–style square look**, probably because those pages have favicons using that style in their `.ico` or `.png`.

## Checklist To Get It Working

1. ✅ Create `favicon.ico` (plus optional PNGs)
2. ✅ Save them in your site folder (e.g. `/assets/icons/`)
3. ✅ Add correct `<link rel="icon"...>` tags in your HTML `<head>`
4. ✅ Clear your browser cache or test in incognito
5. ✅ Save the bookmark again after updating
